# Flexicmd
**Flexicmd** — это проект на C++17, демонстрирующий обёртку функций произвольной сигнатуры (`Wrapper`) и динамическое исполнение команд через `Engine`.  
Проект включает юнит-тесты для проверки корректности работы различных методов.
Вы можете добавлять свои функции(src/Subject.cpp) и тесты(tests/TestCases.h) для них.

## Требования
- 64-битная операционная система
- Компилятор с поддержкой C++17 (MSVC, GCC, Clang)
- CMake >= 3.10
- Visual Studio 2022 (опционально - можно использовать любой другой генератор)

## Структура проекта
flexicmd/
|
├─ include/ — заголовочные файлы
|
├─ src/ — исходники приложения
|
├─ tests/ — юнит-тесты
|
├─ build/ — папка для сборки CMake
|
└─ CMakeLists.txt

## Сборка и запуск программы:
Создайте папку для сборки в корне проекта:

cd build

Соберите через генератор Visual Studio или свой:

cmake .. -G "Visual Studio 17 2022" -A x64

cmake --build . --config Debug

cmake --build . --config Release

Выберите желаемую версию приложения:

cd Debug | cd Release

Запустите приложение:

Flexicmd.exe

## Debug и Release
В Debug режиме выводятся результаты всех юнит тестов

В Release режиме выводятся только проваленные юнит тесты (если такие есть)

## Warning
Во время сборки Release версии появляется warning C4101: e: неиспользованная локальная переменная.

Это нормально. Так как e - возвращаемая ошибка, которую мы не выводим в Release версии.
